# Booly_ai_agent_grok
#Installation
1) Create a .env file
2) Write in it:
Keys=your_grok_api_key
You can use a maximum of 5 api keys.

#Usage
This ai agent has 6 tools. You can view and edit the tools in the tools.py file.
The original code uses the llama3.3 70b-versetile model.
To see the models you can use, go to this link: https://console.groq.com/settings/project/limits
